# LogiTrack

LogiTrack es un MVP académico orientado a la gestión y seguimiento de envíos.

## Estructura del proyecto
- /docs: documentación del proyecto
- /src: código fuente
- /tests: pruebas y scripts

## Flujo de trabajo
La rama main representa el estado estable del proyecto.
Todo desarrollo se realiza en ramas separadas y se incorpora a main
mediante Pull Requests.
